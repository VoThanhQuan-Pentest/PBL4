1. Đọc `01-cau-truc-chuong-trinh.md` để nắm toàn bộ luồng request và thư mục.

3. Đọc từng file trong `thuat-toan/` theo nghiệp vụ cần trình bày.
4. Đọc từng file trong `cong-nghe/` để hiểu khái niệm và cách Nginx, Redis, HikariCP được dùng trong chương trình.
